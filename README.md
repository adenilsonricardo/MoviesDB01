# MoviesDB01

## Sobre
Aplicação utilizando uma API do site The Movie Database com chave de acesso, para gerar uma tela com uma lista de filmes de acordo com o endpoint escolhido.

## Objetivo
- Consumo de uma API;
- Validar acesso há API através de uma chave;
- Lista de objetos de um endpoint;
- Mostrar esses dados em uma RecyclerView;
- Montar a estrutura do App combinando arquiteturas Clean e Mvvm

## Ferramentas utilizadas
- Kotlin
- Clean+Mvvm Architecture
- RecyclerView
- API
- DataBinding 
- Retrofit
- Picasso
- Gson
- Koin
